## このプロダクトについて
Clash of Clans というゲームで施設やラボ施設でユニット・呪文のアップグレードの終了時間を計算、保存するものです

## 開発環境

ライブラリをインストール
```bash
npm install
# or
yarn install
```

サーバーの立ち上げ
```bash
npm run dev
# or
yarn dev
```

[http://localhost:3000](http://localhost:3000) でローカルで確認することが可能
